# Problem Statement
Find minimum value to be assigned to the elements so that sum becomes greater than initial sum.

### Samaple Input :
4 2 1 10 6

### Sample Output :
5

### Explanation :
Sum of original array = 4+2+1+10+6 = 23
Sum of modified array = 5+5+5+5+5 = 25
